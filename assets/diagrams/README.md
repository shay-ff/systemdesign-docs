# Diagram Library

This directory contains standardized architecture diagrams, icons, and templates for the System Design Learning Guide.

## Directory Structure

```
diagrams/
├── icons/              # Component icons and symbols
├── templates/          # Reusable diagram templates
├── patterns/           # Common architecture patterns
└── style-guide.md      # Visual design standards
```

## Usage Guidelines

1. **Consistency**: All diagrams should follow the established color scheme and styling
2. **Clarity**: Use clear labels and consistent iconography
3. **Scalability**: Design diagrams to be readable at different sizes
4. **Accessibility**: Ensure sufficient color contrast and alternative text

## Supported Formats

- **Mermaid**: For flowcharts, sequence diagrams, and simple architectures
- **PlantUML**: For detailed system architectures and UML diagrams
- **SVG**: For custom icons and illustrations

## Color Palette

See `style-guide.md` for the complete color palette and usage guidelines.
